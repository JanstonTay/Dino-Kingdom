## Section A: Project Objective & System Overview

The goal of this project is to help you apply what you've learned by building a **Wellness
Challenge Web Application**. This app builds on your backend and gamification features
from CA1. Users will earn points by completing wellness challenges, which can be used to
unlock gamified experiences such as quests, rewards, or badges.

Your focus in this assignment is on **frontend development** , integrating it with your existing
backend from CA1. You’ll enhance the user experience by building interactive features and
extending functionality.

**Requirements:**

1. **Registration & Login Pages**
    Design simple, user-friendly pages for signing up and logging in.
2. **Authentication Flow**
    Implement secure frontend authentication. Use **JWT** for session management and
    **Bcrypt** for password hashing on the backend.
3. **Wellness Challenge Management**
    Allow users to:
       o Create wellness challenges
       o Track progress and completion
       o Earn and view points
       o Interact with your gamification features from CA1 (Quests, Badges etc)
4. **Review System**
    Create a page where users can:
       o Give **1–5 star reviews** on challenges or completed activities
       o **Edit or delete** their own reviews
       o **View all reviews** from other users

Below are the key competencies that will be assessed:

1. **Backend Server** (10 Marks)

2. **Functionality** (10 Marks)
